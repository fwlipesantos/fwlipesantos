# Desktop Developer | Java & C#

Sou um desenvolvedor com conhecimento em **aplicações desktop**, utilizando **Java** e **C#**. Tenho explorado **JavaFX**, **Swing**, **WPF** e **WinForms** para criação de interfaces e estou em processo de aprendizado sobre **multithreading**, **integração com APIs** e **automação de processos**. Também estou focado em aplicar boas práticas como **Design Patterns** e **otimização de performance** nas soluções que crio.

Além disso, possuo conhecimentos em **análise de dados** utilizando **Java**, com ênfase em técnicas de extração, manipulação e visualização de dados.
